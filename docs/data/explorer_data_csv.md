# explorer_data.csv

## Generate

No generation script. This is a flattened CSV export of `explorer_data.json`, likely produced manually or in a notebook.

## Description

The same data as `explorer_data.json` but in CSV format with pathway scores and variance fractions as separate columns (e.g. `pathway_0_score`, `pathway_0_variance_fraction`). Contains 5,700 reviews (2,700 train + 3,000 test), one per row.
